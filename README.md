# @juicerq/template

SOTA web template — Bun + Hono + oRPC + Arktype + Drizzle + React 19 + TanStack Router + TanStack Query + Jotai + shadcn + Tailwind v4.

**Run:** `bun install && bun db:up && bun db:migrate && bun dev`

**Build:** `bun build`
